# 🏢 工运 — 打工人每日运势

> 不看星座不看八字，看你的工位朝向算今日工运！

## ✨ 特色

- 🧭 根据工位朝向、位置、入职日、工号生成每日运势
- 📊 五维工运：摸鱼运 / 汇报运 / 加薪运 / 甩锅运 / 午饭运
- 📜 每日宜忌、幸运物、打工人金句
- 🔄 每日自动刷新，同一天同一组输入结果一致
- 📱 移动端优先，暗黑星空风格

## 🚀 部署

### Vercel（推荐）

1. Fork 或推送到 GitHub
2. 登录 [vercel.com](https://vercel.com)
3. Import 仓库 → 自动部署完成

### GitHub Pages

1. Settings → Pages → Source 选 `main` 分支
2. 自动生成 `https://你的用户名.github.io/gongyun`
